### 按表达方式分类
1. 声明式语言
声明式语言隐藏了执行细节，是比较高层次的描述方式，我们通常只告诉程序我们需要什么结果。比如如下的语言和例子：

```
JSON,HTML,SQL,CSS,LISP,Clojure，Haskell
```
例子

```sql
SELECT * from user
WHERE user_name = Ben
```
1. 命令式语言
命令式编程（Imperative）：详细的命令机器怎么（How）去处理一件事情以达到你想要的结果（What）

```
C, c++, PYTHON,C#,JS,JAVA
```
