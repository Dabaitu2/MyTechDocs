# 一些概念
Tarball 就是Linux的一种打包工具，简单说来就是压缩解压的
一系列命令
c： 创建备份（Create）
v： 表示回显模式（Verbose），将产生一系列备份的文件列表
f： 指定使用的备份文件，它后面的参数为输出的备份文件名
x： 展开备份文件
t： 显示备份文件的内容
```bash
# 把所有/etd的文件打包成backup.tar
$ tar cvf backup.tar /etc 
```

### chmod 777 是啥意思
因为文件的权限表示为
rwxrwxrwx 这样的格式
表示文件所有者，文件所有者所在组成员，其他用户
对此文件的权限，r可读，w可写，x可执行
